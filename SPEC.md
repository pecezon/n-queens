# N-Queens GA - Specification

## Descripción del Proyecto
Resolución del problema N-Queens en Python utilizando Algoritmos Genéticos (GA). El proyecto demanda un análisis comparativo entre dos variantes del algoritmo y un componente visual de los resultados.

## Tareas y Requerimientos

### 1. Implementación de Algoritmos Genéticos
- [ ] **Versión 1**: Diseñar y programar un método de selección, cruzamiento y mutación.
- [ ] **Versión 2**: Diseñar una configuración alternativa con distintas decisiones de diseño.
- [ ] Asegurar el soporte para distintos tamaños de tablero requeridos (ej. N=8, 16, etc.).

### 2. Análisis y Rendimiento
- [ ] Implementar un script de validación que corra 30 veces cada una de las dos versiones.
- [ ] Medir y registrar el tiempo transcurrido hasta encontrar el tablero óptimo para cada ejecución.
- [ ] Calcular y reportar el promedio de tiempo por cada algoritmo.

### 3. Visualización y Entregables
- [ ] **Componente Visual**: Desarrollar la visualización del tablero mostrando la solución óptima sin ataques.
- [ ] **Gráficas**: Generar gráficas descriptivas del comportamiento evolutivo y tiempos de ejecución.
- [ ] **Reporte Final**: Redactar reporte que incluya: Introducción, tabla descriptiva de diseño, análisis de gráficas y conclusiones.
- [ ] **Presentación**: Preparar diapositivas de resumen para exposición de 10 minutos.
